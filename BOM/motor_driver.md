# L298N Motor driver

 Dimensions: ‎ 5 x 5 x 3 cm

Weight: 27 grams 

Motor output voltage	5V – 35V
Motor output voltage (Recommended)	7V – 12V
Logic input voltage	5V – 7V
Continuous current per channel	2A
Max Power Dissipation	25W
